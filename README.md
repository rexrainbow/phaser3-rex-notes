## Demo

### Prepare

1.  install [node.js](https://nodejs.org/en/)
2.  Click `install.bat`
    -   `npm install`

### Run examples

1.  Click `*.bat` in folder `examples`, it will open application on browser

## File structure

-   Folder `plugins` : source code of plugins
-   Folder `examples` : examples
-   Folder `docs` : notes
